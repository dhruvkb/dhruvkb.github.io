\subsection{About}
Updating a résumé everytime you accomplish or create something new is daunting,
especially when you have to maintain two versions of it for different roles.

\subsection{Attributes}
\begin{itemize}
  \item Developed the résumé using LaTeX.
  \item Résumés for both roles can be generated using the same codebase.
  \item Uses minimal dependencies, nothing outside of TeX Live.
  \item Produces beautifully typeset outputs in two themes.
  \item Clean, well-documented, easy-to-fork and easy-to-modify code.
\end{itemize}
